# WebAutomation

### Usage
        $ git clone https://github.com/ryallurkar/PageObjectPattern.git
        $ cd PageObjectPattern
        $ mvn clean test

### Reporting
 - [Allure Reporting](http://allure.qatools.ru/)

- Open Terminal > go to project directory > run 


        $ allure serve allure-results
